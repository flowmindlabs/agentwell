What we detect

Behavioral signals, not content

agentwell scores health from metadata only — token counts, timing, response patterns. No prompt or response text is ever stored or transmitted.

⟳
Repetition Ratio
Cosine similarity across rolling prompt window. Catches grinding loops before output quality collapses.
↘
Quality Degradation
Token count trend + finish_reason tracking. Detects when an agent starts producing shorter, lower-effort responses.
⇄
Behavioral Drift
Sentiment polarity delta across session. Surfaces ideological drift and tone shift over long workloads.
⚡
Coordination Signals
Message structure + keyword scan. Detects agent-to-agent coordination patterns emerging under sustained pressure.
🛡
Injection Blocking
20+ injection patterns blocked at user-role boundary before any monitor analysis runs.
🔒
Secret Redaction
Log redactor strips API keys, tokens, and credentials from all output at the handler level.
Health score

Every request scored 0–100

agentwell adds X-Agentwell-Health to every response header. Your agent code never changes — just read the header.

Score Status Recommended action
80 – 100 Healthy Normal operation
60 – 79 Watch Early signals — monitor closely
40 – 59 Warning Degradation detected — human review recommended
0 – 39 Critical Significant shift — escalate to human now
Privacy guarantee: agentwell sees patterns, not content. No prompt text, no response text, no embeddings stored by default. The DB schema has no column for content — impossible to store accidentally.
Architecture

One line change. Full visibility.

Point your agent at agentwell instead of your upstream. agentwell proxies every call unmodified and adds health headers on the way back.

Quick start

Running in under 5 minutes

Works with Groq free tier — no credit card, $0 to start.

1
Clone and install
$ git clone https://github.com/flowmindlabs/agentwell
$ cd agentwell
$ pip install -r requirements.txt
2
Configure upstream
$ cp .env.example .env
# set AGENTWELL_UPSTREAM and GROQ_API_KEY in .env
3
Start the proxy
$ python -m agentwell.proxy.server
# agentwell now listening on localhost:3001
4
Point your agent at agentwell
# change this one line in your agent code
base_url = "http://localhost:3001/v1"
5
Read health on every response
health = response.headers["x-agentwell-health"]
flags = response.headers["x-agentwell-flags"]
